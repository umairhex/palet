# PALET

PALET is a minimal design asset manager for storing color palettes and font files.

Upload `.ttf` fonts, manage palette tokens, and maintain design assets in one simple authenticated dashboard built with Vue 3 and Vite.

---

## Features

- Secure authentication
- Upload and manage font files (`.ttf`)
- Create and edit color palettes
- Centralized design asset storage
- Clean dashboard UI
- Ready for deployment on AWS

---

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Pinia
- Clerk Authentication
- AWS
